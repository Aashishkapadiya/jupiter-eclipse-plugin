This file introduces you to the major components of the Jupiter plug-in pacakge. It includes information on the contents of the package you have downloaded, instructions on how to install the plug-in into Eclipse, and disable and/or unistall it from the Eclipse.

Contents of this package

 

Directory Contents
. Top-level directory files include:
  • InstallationGuide.html (this document)
  • History.html contains the version history of Jupiter
./plugins Contains Jupiter plug-in:
  • @PLUGIN_NAME@.@QUALIFIER@  contains the jupiter jar file to be invoked by Eclipse
./features Contains Jupiter plug-in feature:
  • @PLUGIN_NAME@.@QUALIFIER@  contains the feature file to be manipulated by Eclipse (e.g. configuration to disable and/or unisntall Jupiter plug-in)

Installation

These instructions assume that you have unzipped the zip file to create a directory containing the plug-in components.

  1. Please make sure the following before the installation process:

    1. The 1.4 or above level Java runtime or Java development kit (JDK) is installed on your environment.

      If you are using the previous version of the 1.4, You are supposed to install the 1.4 or above and you can specify "vm" optoin to start up your eclipse with the 1.4.   (NOTE: Compatible with JDK5.)
    2. Your Eclipse version is suitable for Jupiter @QUALIFIER@ stream.

      You are supposed to select the proper version. Note that Eclipse had big change of the API from 3.0 Millstone 6 so that the @QUALIFIER@ Eclipse users should use @PLUGIN_NAME@ whose qualifier is @QUALIFIER@.
    3. Your Eclipse is shut down.

  2. Copy both ./plugins/@PLUGIN_NAME@.@QUALIFIER@ and ./features/@PLUGIN_NAME@.@QUALIFIER@ into <installed_eclipse_dir>/plugins and <installed_eclipse_dir>/features respectively.

    The ./features/@PLUGIN_NAME@.@QUALIFIER@ helps you to configure Jupiter plug-in such as disablity and/or unistallation of it gracefully. Othewise you might see some error messages in the error log of Eclipse because the Jupiter related view and action button would be still stored in the workbench just after the Jupiter plug-in deletion.

  3. Start your Eclipse.

  4. Show "Open Jupiter Issue View" button in either way:

    1. Select "Window | Reset Perspective" in the main tool bar.

    2. Select "Window | Customize Perspective", check "Jupiter" (the check box is located in "Commands" tab.

  5. Click the appeared button to open Jupiter Issue View.

Uninstallation

To unistall Jupiter plug-in, please follow the instrcution carefully. Otherwise you might have some error messages in the error log of the Eclipse.

2x stream users (excludes the users who use Eclipse 3.0 until M5)

  1. Make sure all Jupiter related extensions are closed:

    1. Close Jupiter view by unchecking "Jupiter" in the "Window | Customize Perspective | Window > Show View ".

    2. Close Jupiter action button by unchecking "Jupiter" in the "Window | Customize Perspective | Other".

  2. Shut down your your Eclipse.

  3. Delete the following directories and file:

    1. Delete the @PLUGIN_NAME@.2x directory in the <installed_eclipse_dir>/plugins directory.

    2. Delete the @PLUGIN_NAME@.2x directory in the <installed_eclipse_dir>/features directory.

    3. Delete the csdl.jupiter directory in the <installed_eclipse_dir>/workspace/.metadata/.plugins directory.

NOTE: We no longer provide support for Eclipse 2.x version. Upgrade your Eclipse, Buddy! 

3x stream users (3.0 and 3.1)

  1. Make sure all Jupiter related extensions are closed:

    1. Close Jupiter view by unchecking "Jupiter" in the "Window | Customize Perspective | Window > Show View ".

    2. Close Jupiter action button by unchecking "Jupiter" in the "Window | Customize Perspective | Other".

  2. Select "Help | Software Updates | Manage Configuration...", click the installed "Jupiter for Eclipse 3.x stream..." traversing "Eclipse Platform | file:...".

  3. Select the "Disable" link to follow the instrcution, and restart your Eclipse.

  4. Select "Help | Software Updates | Manage Configuration...", click "Show disable features" button in the third icon from the left of the main icon bar in order to see the disable Jupiter plug-in.

  5. Select it and "Unistall" link to follow the instrcution, and restart your Eclipse.
