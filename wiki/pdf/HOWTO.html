How To Generate Jupiter User Guide PDF

Please note this script is meant for Linux/Unix. There is no Windows version.

  1. From the Jupiter SVN repository checkout the wiki folder. See http://code.google.com/p/jupiter-eclipse-plugin/source/checkout for information on accessing the Jupiter SVN repository.
  2. Make sure Perl is installed and available. The easiest way to install Perl is to use the ActivePerl distribution at http://www.activestate.com/Products/activeperl/index.mhtml.
  3. Make sure that Java is installed and available. Java can be obtained from http://java.sun.com.
  4. Install wt2db (WikiText to DocBook). Download the latest version of wt2db from http://tldp.org/downloads/. Follow the instructions in the INSTALL file included in the archive.
  5. Download DocMan (DocBook Toolchain Manager) from http://www.goshaky.com/docman-distfiles/DocMan/. Extract the files to a folder called "docman" and place this folder inside of the wiki/pdf folder.
  6. On the command line change directory to the "pdf" folder inside of the "wiki" folder. Execute the script by typing ./make.pdf.sh. The script does not work if not executed while in the wiki/pdf folder. The generated PDF will be in the wiki/pdf folder.
Directory Structure: 
        wiki/
            UserGuide.wiki
            pdf/
                footer.xml
                header.xml
                HOWTO.html
                make.pdf.sh
                ug2pdf.docman
                UserGuide.pdf (generated after following the directions above)
                docman/  (created in the directions above)
